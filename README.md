# Reactor Agent

[![Build Status](https://travis-ci.org/thockin/go-build-template.svg?branch=master)](https://travis-ci.org/thockin/go-build-template)

Reactor Agent is a simple daemon application and CLI that instantly transforms your server into a development environment with all the tools you need to instantly start developing applications from anywhere.

## Installation

### Ubuntu

To install reactoragent on Ubuntu you can use `snap` to install.

```bash
sudo snap install reactoragent
```

### MacOS via Homebrew

```bash
brew install reactoragent
```

## Features

(\*) feature is still in development.

- If your development server's IP constantly changes, reactoragent can keep it up to date for you.
- Browser Based IDE\*
- Built in Continuous Integration\*
- Instant Development Databases\*
- Instant Hosting Environments (dev, test and production) for your Web Applications and APIs\*
